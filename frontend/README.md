# Support CRM - Intern Assessment Project

A professional, full-stack Support CRM application built to handle customer tickets efficiently.

## 🚀 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, SQLite
- **Key Features:** Real-time ticket searching, status filtering, and an AI-enhanced tracking system.

## 📁 Key Project Structure
```text
frontend/
├── src/
│   ├── components/    # Reusable UI parts (SearchBar, TicketCard)
